Hexagon shapes fix each main island's size exactly (the land-share slider is then ignored). Gold fields only ever spawn on outer islands. Islands are always placed just one sea tile apart from their nearest neighbour where space allows.

Ports

Classic positions fix WHERE the triangles sit (the official spots on 19-hex islands); the distribution setting still decides WHICH port goes in each spot. One port per sea tile, no hex touches two ports. Ports whose triangle touches a foggy tile show "?" until it's revealed.

Fog of war

In edit mode use the ☁ Fog tool to toggle covers on any tile. In play mode click a covered tile to reveal it. Fully covered main islands keep a gold coastline outline.

Save / load

Saved defaults load automatically every time you open the app. The table and collection sections have their own save buttons if you only want to remember those.
